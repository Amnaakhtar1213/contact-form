## CONTACT-FORM

A Simple contact-form build with only *HTML* and *CSS*.The user fill the form by giving the details like name, email, phone, subject and messages also.It also has a send button and all data display the bottom of the form.

---------------------------

## FEATURES

- Enter name
- Enter Email
- enter Phone
- Enter you message
- A display section where all info display .

--------------------------

## TECHNOLOGY

1. **HTML**
2. **CSS**
